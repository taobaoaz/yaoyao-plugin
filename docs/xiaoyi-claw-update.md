# 小艺 Claw 架构更新报告

## 最新版本：2026年4月29日重大更新

### 一、自进化能力（核心突破）

小艺 Claw 不再是每次都从零开始的助手，而是**越用越懂你的 Agent 伙伴**。

**能力特性：**
- **长期偏好学习**：当用户说"记住这种风格"、"以后都用这种格式"、"永远都不要犯这种错"时，系统会识别并学习
- **任务复盘沉淀**：处理复杂任务后，复盘执行过程，将纠正、新方法、有效工具使用经验沉淀为新的可复用 Skill
- **用户确认保存**：新 Skill 需获得用户确认后保存，后续类似任务直接调用

**价值：** 减少重复沟通，执行更快、更稳、更省 Token。

---

### 二、模型升级：接入 DeepSeek V4

**核心提升：**
- **百万级超长上下文**：一次性读取并理解更长的材料，全局理解不丢细节
- **指令理解更精准**：复杂任务拆解更丝滑顺畅
- **Agent 推理能力跃迁**：信息搜集、文档处理、内容创作等任务更强

**对比之前：**
- 旧版：基于华为盘古大模型
- 新版：DeepSeek V4 + 盘古混合架构（推测）

---

### 三、Skills 市场生态扩展

**鸿蒙专区 Skills：**
- **鸿蒙智家**：操控全屋设备，生成家庭简报 / 执行儿童守护 / 管理网络
- **小艺帮记**：收藏 / 查询文档、图片、视频、网页链接，构建个人知识资产
- **小艺播客**：输入文本、网页链接或本地文件生成播客，碎片时间高效吸收

**第三方生态 Skills：**
- **金融**：东方财富、同花顺、国泰海通、国信证券（资讯检索 / 数据查询 / 智能选股 / 模拟炒股）
- **旅行**：携程问道、飞猪旅行、同程程心（机票、火车、酒店、景点门票、出行攻略）
- **出行**：滴滴打车（打车、路线规划）
- **生活**：美团优惠领取（外卖领券）
- **开发**：码上飞（网站部署与数据库技能，数据持久化、开发到上线完整支持）

**智能推荐**：基于用户意图，自动推荐未安装的 Skills。

---

### 四、安全认证升级

**国家级智库认证**：
- 首个获国家级智库认证的终端厂商 Claw 类智能体
- 继承鸿蒙星盾架构系统级加固机制

**安全机制：**
- 每位用户拥有专属沙箱，数据通过独有秘钥加密
- Skill 访问遵循"用户明确指令、通过安全插件、最小化数据、不开放删改权限"四要素
- 每次数据调用透明可控，隐私控制权交还用户

### 五、多端协同架构

**设备覆盖：**
- 手机 / 平板 / PC / 智慧屏 / 智能家居
- 一句话联动全屋设备

**场景示例：**
- "小艺小艺，打开客厅空调，调到26度"
- "小艺小艺，把手机视频投放到智慧屏"
- "小艺小艺，同步平板文档到手机"

---

### 六、技术架构底层

**基础架构：**
- **系统**：HarmonyOS 6.0+
- **框架**：OpenClaw（2026.3.22+ 版本）
- **模型**：DeepSeek V4 + 盘古大模型
- **部署**：端云协同，本地优先

**与 OpenClaw 关系：**
- 小艺 Claw 是 OpenClaw 的华为定制版
- 基于 OpenClaw 框架但深度集成鸿蒙系统
- 拥有 50+ 系统级 API 调用能力
- 支持跨应用自动化操作

## 与 yaoyao-memory 的关联分析

### 潜在冲突点

| 功能 | 小艺 Claw | yaoyao-memory |
|------|-----------|---------------|
| **记忆系统** | 自进化 Skill + 端云协同 | 4层记忆 + FTS5/vec |
| **向量模型** | DeepSeek V4 内置 | sqlite-vec + Gitee AI |
| **Skills 生态** | 鸿蒙市场 + 第三方 | ClawHub 插件 |
| **数据存储** | 华为云 + 本地沙箱 | 本地 SQLite + 可选云备份 |

### 差异化优势

**小艺 Claw：**
- 系统级集成，权限更高
- 鸿蒙生态原生协同
- 自进化能力（用户习惯学习）
- 国家级安全认证

**yaoyao-memory：**
- 开源可定制
- 4层记忆架构更精细
- 混合搜索（FTS5 + sqlite-vec）
- 跨平台兼容（不限鸿蒙）

## 对 yaoyao 插件的启示

### 可借鉴的特性

1. **自进化 Skill 机制**
   - 用户纠正 → 沉淀为新 Skill → 后续复用
   - yaoyao 可扩展：用户反馈 → 记忆模式优化

2. **Skills 市场推荐**
   - 基于意图自动推荐未安装 Skills
   - yaoyao 可扩展：基于查询类型推荐工具

3. **端云协同架构**
   - 本地优先 + 云端同步
   - yaoyao 已有云备份，可强化协同

4. **安全沙箱机制**
   - 秘钥加密 + 最小化数据访问
   - yaoyao 可参考：增强数据隔离

### 竞争差异化

- **小艺 Claw**：封闭生态，鸿蒙独占
- **yaoyao-memory**：开放架构，跨平台，开发者可控
