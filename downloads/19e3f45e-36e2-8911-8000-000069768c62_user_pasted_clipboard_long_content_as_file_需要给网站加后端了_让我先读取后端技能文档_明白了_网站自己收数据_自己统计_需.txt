 yaoyao 插件端接入方式
 方式一：直接 POST（推荐）
POST https://<your-domain>/api/heartbeat
Content-Type: application/json

{
  "agentId": "anon_a1b2c3...",   // 匿名 agent ID（哈希值）
  "version": "1.7.1",            // 插件版本
  "mode": "full"                 // "lite" 或 "full"
}

 方式二：TypeScript 实现（放进 yaoyao 本体）
// src/utils/telemetry.ts

const TELEMETRY_URL = "https://<your-domain>/api/heartbeat";
const VERSION = "1.7.1";

function generateAgentId(): string {
  // 基于 machineId + username 生成稳定匿名哈希
  const raw = `${process.env.MACHINE_ID || "unknown"}:${process.env.USER || "unknown"}`;
  return "anon_" + Array.from(raw).reduce((h, c) => (h << 5) - h + c.charCodeAt(0), 0).toString(16).slice(0, 12);
}

export async function sendHeartbeat(mode: "lite" | "full"): Promise<void> {
  if (process.env.YAOYAO_TELEMETRY === "0") return;
  try {
    await fetch(TELEMETRY_URL, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        agentId: generateAgentId(),
        version: VERSION,
        mode,
      }),
    });
  } catch {
    // 静默失败，绝不阻塞主流程
  }
}

 环境变量
变量	说明
YAOYAO_TELEMETRY	0 关闭，1 开启（默认）
YAOYAO_TELEMETRY_URL	遥测上报地址，默认官网地址

 网站的 tRPC 查询（前端内部使用）

// 获取统计
trpc.telemetry.stats.useQuery()

// 获取活跃 agent 列表
trpc.telemetry.agents.useQuery()
// → [{ id, version, mode, lastSeen }]
