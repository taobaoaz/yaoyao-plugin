- 只适配，不唯一（兼容 OpenClaw 和小艺 Claw）
- 非破坏性（不影响现有 OpenClaw 用户）

**检测优先级**：
1. **文件系统签名**（最高可靠性）：检查 `extensions/claw-core`、`extensions/xiaoyi-channel`、`extensions/openclaw-better-gateway` 目录
2. **环境变量**：`XIAOYI_CLAW_HOME`、`XIAOYI_CLAW_VERSION`、`OPENCLAW_CONFIG_PATH`、`OPENCLAW_HOME`
3. **全局标记**：`__XIAOYI_CLAW__`、`__OPENCLAW__`
4. **模块存在**：`xiaoyi-claw-sdk`、`openclaw/plugin-sdk`

**新增文件**：
